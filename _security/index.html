Defensive Policy Notes

Static rules for keeping this security portal useful without leaking operational secrets.

Publish Only Sanitized Summaries

Use this page for public-safe summaries, indexes, and operator navigation. Raw logs belong off-repo.

No Secrets in Static Pages

Never commit tokens, private endpoints, credentials, passphrases, keys, cookies, or auth records.

Separate Public and Private Telemetry

Expose public metrics only. Keep private SOC data behind authenticated infrastructure later.

Document Incidents Carefully

Use plain timelines, impact, remediation, and lessons learned. Avoid publishing exploitable detail.
