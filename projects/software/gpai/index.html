Overview

GPAI is the data+model layer for GP. It provides standardized dataset specs, dataset builders, synthetic data pipelines, evaluation suites, prompt packs, safety classifiers, and fine-tuned model artifacts. Integrations target local inference (GPU/CPU), cloud endpoints, and hybrid orchestrations — all with reproducible training metadata and clear licensing.
