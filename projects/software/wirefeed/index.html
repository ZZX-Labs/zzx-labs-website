WireFeed

A RSS/Atom/JSON news ticker & API reader. Marquee, stack, or grid your headlines. CORS-safe fallbacks, filters, and theming.

Live Ticker Demo

Uses the feeds listed in manifest.json. Some sources may require a proxy due to CORS.

Multi-Source

Mix RSS, Atom, and JSON APIs. Merge, sort, and dedupe headlines.

Marquee/Stack/Grid

Pick the visualization for your wallboard, stream overlay, or newsroom.

CORS-friendly

Optional proxy endpoint included for feeds that block browser requests.

Filters & Rules

Keyword filters, domain rules, and source weighting.

Themeable

Colors, fonts, borders, radius, and light/dark modes via CSS variables.

Embeddable

Drop the widget into dashboards, OBS scenes, or signage pages.

Add Feeds

{
  "feeds": [
    "https://example.com/rss.xml",
    "https://another.com/atom.xml",
    { "type": "json", "url": "https://api.example.com/news" }
  ]
}

Use a Proxy (optional)

{
  "proxy_url": "https://your-proxy.example.com/fetch?url="
}

Server should set permissive CORS headers and stream upstream content.
