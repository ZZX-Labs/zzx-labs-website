Overview

4DV is a toolkit for organizing and visualizing phenomena with a time dimension. It maps entities and interactions into spatial-temporal constructs, enabling “data sculpture”, 4D simulations, and memory reconstructions. It targets research, forensics, simulation authoring, and exploratory UI prototyping.
