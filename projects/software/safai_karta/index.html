What is Safai Karta?

Safai Karta is a tiny, dependable tool for batch-renaming files and sanitizing path segments. It focuses on portability: the same names should work across Linux, macOS, and Windows, and travel safely through archives, clouds, and URLs. Use the UI below or the CLI library to normalize whitespace, strip disallowed characters, collapse duplicates, enforce length limits, and optionally transliterate.

Try It

Options
safe truncated empty → "untitled"

Tip: Put extensions on the last segment. We keep a final extension by default when truncating.

CLI / Library

Install

pip install safai-karta

CLI

safai-karta --replace-spaces _ --windows-safe --lowercase *.mp3
