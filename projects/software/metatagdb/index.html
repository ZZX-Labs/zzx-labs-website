MetaTagDB — Media Tagging & Hash DB

Distributed metadata + content-hash registry for media: verify integrity, tag evidence, and replicate catalogs.

Open Project View on GitHub

All hashing below runs locally in your browser (no uploads).

Overview

MetaTagDB is a media tagging system and hash-indexed database. It produces content-addressable records (SHA-256, size, MIME, timestamps, and user tags) that can be merged, signed, and replicated across peers. Use it for authenticated leak catalogs, forensics triage, deduplication, and cross-set reconciliation.
