ISAN — Internet Streaming Audio Network

Decentralized P2P audio/radio streaming system & platform. Local-first ingest, verifiable manifests, and swarm distribution.

Open Project View on GitHub

This page includes a local demo player (HTTP streams). P2P mesh and node services are outlined below.

Overview

ISAN is a decentralized streaming fabric for radio/audio. Stations publish signed manifests and chunks; listeners fetch via swarms (WebRTC/QUIC) or fall back to HTTP origin. The browser demo below plays standard streams (MP3/AAC/Opus & native HLS where supported) and shows mock swarm metrics for illustration.

        Architecture (Planned)

        • Publisher nodes: ingest audio → chunk → sign → announce via DHT.
        • Swarm transport: WebRTC/QUIC data channels; HTTP fallback via origin and edge caches.
        • Verifiable manifests: station.json (identity, pubkeys, codecs, encodings, fallback URLs).
        • Payments (optional): LNURL/WebLN tipping and per-chunk sats streams (opt-in).
        • Privacy: no tracking beacons; peer IDs are rotating and minimized.
