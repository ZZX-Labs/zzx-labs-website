Overview

audio-tagger scans folders and audio archives, computes robust fingerprints, analyzes technical and musical features (EBU R128 loudness, tempo/BPM, key estimate, spectral centroid/rolloff, channel layout, bit depth, codec), and writes stable sidecar JSON and (optionally) embedded tags (ID3, FLAC Vorbis comment). It favors deterministic runs, small outputs, and provenance fields for audit.
