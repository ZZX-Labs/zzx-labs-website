URLScraper (Firefox)

One-click extract, dedupe, filter, and export of all links on a page — local-only, privacy-friendly.

What it is

A minimalist WebExtension for Firefox that scrapes URLs from the current tab (anchors, iframes, media sources, background CSS URLs, and selectable text), normalizes and de-duplicates them, then lets you export to clipboard, CSV, NDJSON, or a saved text file. Includes pattern filters, domain allow/deny lists, and a simple rules engine for quickly capturing the links you want.

Fast URL Harvest

Grab links from DOM anchors, visible text, media tags, and CSS — all client side.

Filter & Dedupe

Regex filters, domain allowlists/denylists, same-host only, and automatic de-duplication.

Easy Export

Copy to clipboard or export as CSV/NDJSON/plain-text. Zero external calls by default.

Temporary Install (Dev)

2) Click "Load Temporary Add-on..."
3) Select the add-on's manifest.json
4) Pin the toolbar button → click to scrape
