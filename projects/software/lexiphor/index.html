Overview

Lexiphor is a PyQt5 GUI formatter focused on pragmatic consistency for large codebases. It supports C, C++, Java, JavaScript, TypeScript, Python, HTML, CSS, JSON, Go, Perl, Lua, R, Ruby, Rust, and more. Core features: indentation unification, trailing-whitespace cleanup, newline policy, basic AST-signals, and project-level presets.
