What is Speciedex?

Speciedex models biological and mythological taxa, traits, and relations. It stores canonical ranks, supports custom hierarchies, and lets you classify new entries using local heuristics or pluggable ML. It exports clean JSON/CSV for downstream explorers (e.g., SpeciedexExplorer) and inference kernels (SpeciedexCore).
