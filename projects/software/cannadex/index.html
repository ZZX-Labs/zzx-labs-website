Overview

Cannadex is a registry + exchange layer for licensed cultivators, processors, distributors, and retailers. It ingests verified COAs, tracks inventory lots, enforces jurisdictional rules, and exposes a pricing/availability API. Optional settlement modules support ACH, stablecoins, or Bitcoin/LN where lawful. Built to be auditable, API-first, and region-aware.
