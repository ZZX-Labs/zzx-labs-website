ZZX-Core

Software Server Stack · Portable container + GUI system that hosts all web backends. One-click reverse proxy, TLS, databases, queues, observability, and app runtimes — locally or on servers.

What ZZX-Core Provides

Runtimes

Node/Express, Python/Uvicorn, Go, PHP-FPM, Static sites; plug-in templates and health checks.

Reverse Proxy + TLS

Traefik / Caddy / Nginx frontends; automatic HTTPS via Let’s Encrypt, DNS/HTTP challenges.

Data Layer

Postgres, Redis, MinIO (S3), RabbitMQ/Kafka options; backups and persistent volumes.

Observability

Prometheus, Grafana, Loki, Tempo, cAdvisor; pre-wired dashboards and alerts.

Portable by Design

Docker Compose bundle generation, .env, systemd unit; dev → staging → prod.

GUI Controls

Start/stop services, tail logs, inspect metrics, edit config, and roll back safely.
