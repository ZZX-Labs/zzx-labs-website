Overview

Character Generator is a local-first toolkit for fast, consistent character creation. Define archetypes and constraints, roll or curate traits, generate biographies and dialogue samples, and export clean JSON sheets for engines or design docs. Optional model hooks create portrait concepts and voice samples. Versioned templates ensure continuity across seasons, campaigns, or series bibles.
