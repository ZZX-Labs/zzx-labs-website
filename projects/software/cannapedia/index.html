Overview

Cannapedia is a reference knowledge base for cultivars, lineage, morphology, and verified chemistry (COA-backed). It normalizes lab data, captures provenance and synonyms, models terroir/environmental factors, and provides a graph-powered API for discovery—useful for research, breeding, retail education, and compliance reporting. Privacy-respecting; COA ingestion requires signatures from licensed labs.
