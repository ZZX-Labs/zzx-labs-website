ProtoTag

Prototype video tagging workbench used to incubate ideas for MetaTagDB. Add time-coded markers, attributes, hashes, and exports for downstream analysis.

Get Started GitHub Downloads

ProtoTag is a prototype/research tool; expect sharp edges. Its data model informed MetaTagDB’s schema.

Time-coded Tagging

Create instant, range, and category tags with millisecond precision. Annotate events, scenes, faces, or objects.

Sidecar + Export

Non-destructive sidecar (.prototag.json) plus export to MetaTagDB-compatible JSON/NDJSON.

Forensic Hashes

Compute SHA-256 for files and optional rolling frame/sample hashes for verification and deduping.

Structured Schemas

Tags include type, who/what, where, confidence, and evidence URLs.

Keyboard-First

Low-latency hotkeys for play/pause, jog, mark-in/out, new tag, and jump to next/prev tag.

Privacy-Aware

Local-first: analysis runs on your machine. No automatic uploads. Export only what you intend to share.

Get Started

  1. Download a release (or run from source).
  2. Open a video file (MP4, MKV, MOV…) or paste a local/HTTP URL.
  3. Use hotkeys to insert tags (N for new, I/O for in/out, J/L jog).
  4. Save to sidecar (video.prototag.json), then Export for MetaTagDB ingestion.
