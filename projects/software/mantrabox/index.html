MantraBox

A TTS mantra voice synthesizer & looper with beat-sync, cycles, and spatial/binaural layers.

What it is

MantraBox generates and loops spoken/toned mantras using on-device or server TTS. Cue phrases, set tempo, define cycles (e.g., 108), pan/space the voice, and add options like solfeggio layers, drones, or subtle binaural beats. Export stems, share loop “presets,” and stream sessions to ISAN/ISTV.

Multi-Voice TTS

Pick voices (female/male/neutral), timbre, formant shift, whisper/chant styles.

Cycles & Counts

108/54/27 cycles, fixed timers, or infinite; automatic gap/silence shaping.

Beat-Sync

Lock to BPM, tap tempo, or free-run; swing, accent groups, and downbeat markers.

Spatial & Binaural

Stereo drift, L/R call-response, binaural offsets, drone/sine/OM companions.

Presets & Stems

Save/share presets as JSON, export WAV/FLAC stems for DAWs and live sets.

Local-First

Runs offline with local TTS backends; optional server mode for remote control.

CLI (local)

mantrabox \
  --text "Om Mani Padme Hum" \
  --voice "neutral-soft" \
  --cycles 108 \
  --bpm 72 \
  --binaural 7.0 \
  --export ./out/session

JSON preset

{
  "text": "Gate Gate Paragate Parasamgate Bodhi Svaha",
  "voice": "alto-warm",
  "bpm": 64,
  "cycles": 54,
  "layers": { "drone":"C#2", "solfeggio":"396Hz" },
  "space": { "width": 0.35, "drift": 0.1 }
}

Server mode

mantrabox serve --bind 127.0.0.1:7770
# POST /render { preset: {...} }
