VLC-Alarmclock

Schedule alarms that play media through VLC: local files, playlists, or radio streams — with fade-in, weekday rules, and sleep timers.

What it is

VLC-Alarmclock is a small scheduler that launches VLC with your chosen media at set times. Wake up to a playlist, a streaming URL, or a calming soundscape. Define recurring alarms (weekdays, weekends, custom cron-like rules), ramp the volume smoothly, and auto-stop with a sleep timer. Works offline with local files; supports network streams when online.

Fade-in & Volume Curves

Start gentle. Configure ramp duration, target level, and curve shape.

Weekday Rules

Different schedules for workdays/weekends; quick “skip next” toggles.

Streams or Local

Play folders, M3U/PLS playlists, or radio streams (e.g., ISAN stations).

Sleep Timers

Auto-stop after N minutes; optional gradual fade-out.

JSON & CLI

Keep reusable alarm presets as JSON or use a one-shot CLI command.

Cross-Platform

Designed to run on Windows, macOS, and Linux with VLC installed.

CLI (one-shot)

vlc-alarmclock \
  --time "07:30" \
  --media "/music/wake-up.m3u" \
  --fade-in 45 \
  --volume 70 \
  --days "mon,tue,wed,thu,fri"

JSON alarm preset

{
  "label": "Weekday Wake",
  "time": "07:30",
  "days": ["mon","tue","wed","thu","fri"],
  "media": "/music/wake-up.m3u",
  "volume": { "start": 15, "target": 70, "fadeInSec": 45, "curve": "ease-in" },
  "sleepTimerSec": 3600
}

Stream example

{
  "label": "Morning Radio",
  "time": "08:15",
  "days": ["sat","sun"],
  "media": "https://example-radio/stream.aac",
  "volume": { "start": 20, "target": 60, "fadeInSec": 30 }
}
