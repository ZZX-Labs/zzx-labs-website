Overview

GPGalleryAI (GPGAI) is a SFW showcase layer for the GP ecosystem. It provides creator profiles, feed publishing, collections, tag search, and license/attribution metadata. Works offline-first with optional sync to a shared index, and supports content exported from GP pipelines (images, prompts, captions, credits).
