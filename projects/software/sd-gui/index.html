What is SD-GUI?

SD-GUI is a lightweight, production-oriented front end and controller for Stable Diffusion-class models. It manages model assets, performs health checks, exposes REST endpoints, and provides a clean, low-latency queue for batched inference. Use it locally as a desktop app, or run as a headless server with the same API.
