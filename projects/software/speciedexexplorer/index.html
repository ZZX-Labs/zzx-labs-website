SpeciedexExplorer

Block explorer for the SpeciedexCore chain: browse species assets, lineage graphs, transactions, addresses, and verifiable proof artifacts.

What it is

SpeciedexExplorer is a web UI + API stack for querying the SpeciedexCore blockchain state. Inspect taxonomy-linked assets, follow provenance trails, search transactions/addresses, and render lineage graphs with confidence metrics — all backed by verifiable hashes.

Rich Search

Look up species IDs, asset IDs, txids, addresses, and tagged bundles with fast filters.

Lineage Graphs

Visualize evolution links and evidence weights across time windows and datasets.

Audit-Friendly

Every view references content hashes and exportable manifests for third-party checks.

Previews

Docker (recommended)

docker run -p 8080:8080 -e SDX_CORE_RPC=http://core:8332 zzx/speciedex-explorer:latest

Compose

services:
  core:
    image: zzx/speciedex-core:latest
    ports: ["8332:8332"]
  explorer:
    image: zzx/speciedex-explorer:latest
    ports: ["8080:8080"]
    environment:
      SDX_CORE_RPC: http://core:8332

API

GET /api/v1/asset/:id
GET /api/v1/tx/:txid
GET /api/v1/address/:addr
GET /api/v1/lineage?species=Homo+sapiens
