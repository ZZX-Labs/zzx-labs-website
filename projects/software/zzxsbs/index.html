Site Builder System — generate unified project/portfolio/docs/downloads/papers pages from one manifest (GUI & CLI). This page includes a browser-only mini builder that mirrors the desktop tool’s output.

What ZZX-SBS Does

One Manifest → Many Bundles

Create consistent, themed bundles for projects/<kind>/<slug>/, portfolio, docs, downloads, and papers.

Themes & Libraries

Switch site-wide colors, fonts, and layout templates. Reuse designs across new sites and projects.

GUI & CLI

Use a PyQt5 GUI or a scriptable CLI. This page provides a sandboxed in-browser generator for quick starts.
