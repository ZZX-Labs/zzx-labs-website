Overview

AudioLab is a modular digital audio analysis, synthesis, and performance environment built in Python. It unifies sampling, looping, quantization, tone generation, MIDI I/O, glitch synthesis, and TTS into a cohesive framework supporting both CLI and PyQt5 GUI interfaces.

  • Version: 0.1.0-alpha

  • License: MIT
  • Language: Python 3.11+
  • Mode: CLI / PyQt5 GUI
  • Core Engine: sounddevice, numpy, mido, pyttsx3
