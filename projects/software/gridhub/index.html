Open Project View on GitHub

Local-first. Sandboxed iframes. Share layouts via URL.

Overview

GridHub lets you compose a grid of live web panes from a list of URLs. Snap to presets (1×2, 2×2, 1×3, 2×3…), drag to reorder, store/load JSON layouts, or share a single link containing everything (URLs, titles, layout). Useful for monitoring dashboards, pairing docs with terminals, or watching multiple streams at once.

        • Drag & reorder: keyboard navigation and quick focus switching.
        • Local-first: save layouts to JSON; import later with one click.
