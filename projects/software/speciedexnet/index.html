SpeciedexNet

Network relay & telemetry layer for Speciedex — live peer stats, mempool/events firehose, and exportable metrics for dashboards and alerts.

What it is

SpeciedexNet is a daemon + gateway that connects to SpeciedexCore peers, relays blocks/ mempool/lineage events, aggregates node health, and streams normalized metrics via HTTP, WebSocket, and Prometheus endpoints. Build real-time dashboards, trigger alerts, and feed downstream services with a consistent, cacheable surface.

Relay & Firehose

Low-latency fan-out of blocks, mempool entries, taxonomy/asset updates, and lineage events.

Metrics & Health

Peer/session telemetry, lag, orphan rates, RPC stats, exporter for Prometheus/Grafana.

Filters & Sinks

Topic filters, WebSocket streams, durable sinks (S3/FS), signed snapshots for audits.

Previews

Docker (recommended)

docker run -p 7070:7070 -p 7071:7071 -e SDX_CORE_RPC=http://core:8332 zzx/speciedex-net:latest

Compose

services:
  core:
    image: zzx/speciedex-core:latest
    ports: ["8332:8332"]
  net:
    image: zzx/speciedex-net:latest
    ports: ["7070:7070","7071:7071"]
    environment:
      SDX_CORE_RPC: http://core:8332

API

GET  /api/v1/peers
GET  /api/v1/mempool/summary
GET  /api/v1/health
WS   /ws/stream?topics=blocks,mempool,lineage
GET  /metrics               # Prometheus
