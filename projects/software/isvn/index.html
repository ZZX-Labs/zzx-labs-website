ISVN — Internet Streaming Video Network

Give anyone a 24/7/365 “TV network”: decentralized ingest, signed manifests, swarm distribution with HTTP fallbacks.

This page includes a local demo video player (progressive MP4 / native HLS if supported). No external libs.

Overview

ISVN is a decentralized, programmable video network. Publishers push live/looped schedules or VOD using signed manifests; viewers fetch via swarms (WebRTC/QUIC) with HTTP origin/CDN fallback. The demo below uses the browser’s native <video> capabilities.

        Architecture (Planned)

        • Publisher nodes: ingest (RTMP/WebRTC/file) → transcode → chunk → sign → announce via DHT.
        • Manifests: channel.json defines program schedule, renditions, DRM policy (if any), HTTP fallbacks.
        • Transport: WebRTC/QUIC swarms; HTTP(S) + CDN for fallback.
        • Monetization (opt-in): LNURL/WebLN tips, pay-per-segment, gated channels.
        • Resilience: cacheable segments, rotating peer IDs, minimal telemetry for privacy.
