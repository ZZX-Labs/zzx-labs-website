Parallel Explorer

Multi-panel URL explorer: tile, sync, and inspect multiple websites side-by-side in a grid view.

About

Parallel Explorer is a grid-based web interface that allows researchers, developers, and analysts to view multiple web resources simultaneously. Each URL loads in its own panel, allowing synchronized scrolling, comparisons, and simultaneous inspection of content from multiple sources.

Core Features

  • Multi-URL grid viewer (1×2, 2×2, 3×3, custom layouts).
  • Synchronized or independent scrolling modes.
  • Custom panel resizing and drag-and-drop ordering.
  • Offline/local manifest-driven workspace setup.
  • Lightweight, client-side execution with optional server support.

Usage

Load a JSON manifest of URLs or paste them directly into the app. The system will generate a tiled grid where each panel hosts one resource. This is especially useful for OSINT, research dashboards, or monitoring multiple sources in parallel.
