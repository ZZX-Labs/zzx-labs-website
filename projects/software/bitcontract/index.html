Overview

BitContract provides a recipe book for Bitcoin-native agreements: time/threshold locks, vaults, escrows, and oracle-driven payouts — described in human-readable policy, compiled to Miniscript, and executed via PSBT. It favors transparent, reproducible flows over black-box “smart contracts.”
