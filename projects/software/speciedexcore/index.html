SpeciedexCore

Inference kernel for species ID, evolution mapping, and tokenized outputs — designed with architecture rigor inspired by Bitcoin Core.

What it is

SpeciedexCore is the engine layer of the Speciedex suite: a deterministic, testable inference kernel for species classification, lineage graph building, and verifiable artifact exports. It favors reproducibility and audit trails over black-box magic.

Deterministic Pipelines

Stable models + pinned data transforms produce repeatable classifications across nodes.

Lineage Graphs

Build evolution mappings with uncertainty bounds and explainable edges.

Proof Artifacts

Hash-chained result bundles & signed manifests suitable for long-term archives.

Previews

Install

pip install speciedexcore

Quickstart

from speciedexcore import Inference

inf = Inference(models="sdx-core-1.0")
res = inf.classify(image_path="samples/leaf.jpg")
print(res.top_k(5))

CLI

speciedexcore classify --image samples/leaf.jpg --topk 5 --export out/bundle.sdx
