mempoolspecs — Mempool Goggles Port

A portable, mempool.space-compatible “goggles” UI to visualize fee bands, queue depth, and confirmation horizons.

Open Project View on GitHub

Uses public endpoints if configured; you can point it at your own full node or a mempool API mirror.

Overview

mempoolspecs is a UI component set and lightweight client for mempool observers inspired by “goggles” visualizations. It focuses on fee bands (sat/vB), queue depth per band, projected confirmation windows, and a compact status rail for node/API health. It can run read-only against mempool.space-style APIs or your own.

        About this demo

        The demo queries /api/v1/fees/recommended, /api/blocks/tip/height, and /api/mempool when your API base is set (CORS must allow it). Otherwise, “Load Snapshot” renders local sample data. The full app supports additional feeds and historical playback.
