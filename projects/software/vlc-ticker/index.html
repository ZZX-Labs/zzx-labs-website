VLC-Ticker

A PyQt5 companion ticker that shows live track info (ID3 / stream metadata) from VLC — always-on-top, skinnable, and ultra-light.

What it is

VLC-Ticker reads VLC’s current media metadata (ID3 tags or its stream equivalents) and renders it as a small floating ticker window. Show artist/title, album, station name, or custom templates. Position it anywhere, lock it “always on top”, and tweak font, colors, opacity, scroll speed, and marquee behavior. Ideal for streamers, kiosks, and studio monitors.

Live Metadata

Parses VLC’s now-playing via RC / HTTP API. Falls back to filename heuristics.

Marquee & Static Modes

Auto-scroll long titles, or hard-wrap and resize to fit.

Skinnable

Theme JSON for colors, fonts, corner radius, borders, shadow, and opacity.

Dock & Snap

Snap to screen edges, multi-monitor aware, remember last position.

Hotkeys & Tray

Toggle show/hide, pause scroll, and open settings from the system tray.

UTF-8 & Sanitization

Encoding fixes, emoji-safe, trims repeated station banners & junk prefixes.

VLC HTTP interface

VLC ▶ Preferences ▶ Show settings: All
Interface ▶ Main interfaces ▶ Web (enable)
Lua ▶ Lua HTTP ▶ Password: ********
Restart VLC and keep it running.

Run VLC-Ticker

vlc-ticker \
  --vlc-host 127.0.0.1 --vlc-port 8080 --vlc-pass "********" \
  --template "{artist} — {title}" --marquee --speed 40 --font-size 18

Theme JSON

{
  "fontFamily": "IBM Plex Sans",
  "fontSize": 18,
  "fg": "#e8e8e8",
  "bg": "#0f1216",
  "accent": "#c0d674",
  "opacity": 0.9,
  "radius": 10,
  "border": "1px solid rgba(255,255,255,0.18)"
