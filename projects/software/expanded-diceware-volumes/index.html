Printable volumes + machine-readable JSON/CSV. Consistent 5-dice indices (11111–66666), integrity-checked, and offline-friendly.

Overview

Expanded Diceware Volumes assembles large, well-vetted Diceware wordlists as “books” you can print or query. Sets include multilingual lists, family-friendly subsets, technical terms, and thematic packs. Each entry carries its dice index, checksum, and normalization rules. Generate memorable, robust passphrases with transparent, auditable sources.

        Highlights

        • Volume sets: language packs (EN, ES, FR, HI, etc.), themed sets (science, nature, classic), family-safe variants.
        • Strict indexing: canonical 5-dice (base-6) mapping, no gaps; reproducible lookups with physical dice.
        • Normalization: ASCII/Unicode variants, hyphenation rules, optional diacritics, print-safe typography.
        • Data formats: PDF “books”, JSON, CSV, SQLite — integrity hashes and release manifests.
        • Entropy-aware: length planners, entropy calculators, and guidance for secure dice rolls.
        • Offline-first: everything works without internet; verify signatures and hashes locally.
        • Tooling: CLI helpers to sample passphrases from any volume or merge custom lists.
