CyberChefKit

A localized online/offline container program that runs GCHQ’s CyberChef unmodified — fully offline or served locally — with optional language-mirror toolchains (C, C++, Python, Go, R, Perl).

Open Project View on GitHub

CyberChefKit is not a fork of CyberChef. It runs the official upstream build locally (offline) or via a local/online server. Companion subprograms mirror containerized workflows across multiple languages.

Overview

CyberChefKit packages the official GCHQ CyberChef for dependable, offline-first usage and repeatable pipelines. Use it as a portable desktop app, a LAN-only server, or an internet-facing service (your choice). Around that core, language-specific helpers — CyberChefC, CyberChefCpp, CyberChefPy, CyberChefGo, CyberChefR, CyberChefPerl, etc. — mirror the “containerized recipe + IO” model so teams can integrate CyberChef-like flows directly in their native stacks without modifying CyberChef itself.
