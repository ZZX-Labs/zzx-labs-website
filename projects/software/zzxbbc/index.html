ZZX-BBC (BB Communicator)

In-house LoRa mesh communicator with end-to-end encryption. A BlackBerry-style device and stack (software, firmware, hardware, web) for text, email relay, voice, video, and file transfer across Wi-Fi, Mesh/LoRa, GSM/5G gateways.

Capabilities

Text & Email Relay

E2EE chats across mesh; optional SMTP/SUBMIT relay when a gateway is present.

Voice & Video

Adaptive codecs with store-and-forward fallback; real-time over Wi-Fi/Cell, burst over LoRa.

File Transfer

Chunked, signed, and resumable transfers; opportunistic multi-hop routing.

Mesh Routing

Flood-guarded gossip + distance vector; region-legal LoRa parameters per band.

Keys & Identity

Device seed → X25519+Ed25519 subkeys; QR provisioning, revocation, and trust graph.

OTA & Telemetry

Signed firmware updates with canary; local web console; redaction-first diagnostics.
