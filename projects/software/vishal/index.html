Vishal

TUI + Web dashboard for pipelines & workers. Observe jobs, nodes, artifacts, and throughput in real-time.

What it is

Vishal is a visual shell for operational awareness. It connects to runners like Vikram and shows queues, active/failed jobs, node health, and artifact timelines. Use the terminal UI (TUI) on a server, or the optional web UI for a live wallboard. Zero heavy dependencies; simple JSON event streams in/out.

Unified View

Jobs, queues, workers, and artifacts on one screen with fast filters and fuzzy search.

TUI + Web UI

ncurses-style terminal dashboard + optional browser panel for big displays.

Live Metrics

Throughput, latency, retries, and success/failure rates with compact sparklines.

Pluggable Sources

JSON event adapters for Vikram, VidGhee, VideoSort, GardenHarvester, and custom emitters.

Action Hooks

Rerun, cancel, drain, quiesce worker — with role-checked safety prompts.

Low Ops

Single binary/server, optional SQLite state, exports to NDJSON/CSV.

Run (TUI)

vishal tui \
  --source http://localhost:7788/events.ndjson \
  --since 2h --refresh 500ms

Run (Web)

vishal web \
  --bind 0.0.0.0:8090 \
  --source http://runner.local:7788/events.ndjson \
  --readonly
