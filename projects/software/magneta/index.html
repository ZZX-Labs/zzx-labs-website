Overview

Magneta is a cross-platform, open-source data sanitization toolkit that securely and verifiably erases all storage media types — HDD, SSD, NVMe, SD/microSD, and USB — using a blend of firmware-level and cryptographic overwrites. It meets and exceeds NIST 800-88 and DoD 5220.22-M standards for total data destruction.
