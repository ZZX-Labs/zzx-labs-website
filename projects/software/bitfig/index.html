BitFig

Bitcoin Core & Knots Configuration Builder — generate reproducible, human-readable bitcoin.conf and knots.conf templates with preset bundles for full nodes, pruned nodes, miners, and developers. Features both web-based and local CLI/GUI operation.

Overview

BitFig lets users compose complete node configuration sets for Bitcoin Core and Bitcoin Knots. It supports advanced features such as Tor, I2P, RPC whitelisting, P2P policy tuning, mempool & fee policies, and wallet or mining presets. Outputs are clean JSON or plaintext configs suitable for deployment or version control.
