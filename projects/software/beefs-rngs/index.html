Overview

Deterministic PRNGs and pluggable true-random sources, wired through a common API. Includes stream & block generators (e.g., ChaCha20-based DRBG), entropy harvesting from hardware TRNGs, and verifiable randomness derived from Bitcoin headers. Ships with scripts to run Dieharder, TestU01 (SmallCrush/Crush/BigCrush), and PractRand across seeds, sizes, and platforms. Reproducible test manifests and HTML reports included.
