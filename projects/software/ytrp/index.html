YTRP / YTRPV

Minimal front-end for yt-dlp. YTRP = Audio-first rips (tags, covers, replaygain). YTRPV = Video-first rips (mux, subs, chapters).

Build & Run a Rip

Paste a URL (YouTube, SoundCloud, etc. — anything supported by yt-dlp). Select audio/video mode and options, then generate a command or run via a local API.

Local API not found? The command still works in your terminal if yt-dlp, ffmpeg, and encoders are installed.

Queue & History

Audio-first (YTRP)

Auto-extract audio, select target codec, embed metadata/cover, and normalize loudness.

Video-first (YTRPV)

Mux best video+audio, embed soft-subs, chapters, and preserve originals where possible.

Playlists & Ranges

Rip entire playlists or just segments. Start/end timestamps for precise cuts.

Friendly Templates

Curated output templates so your library stays clean and deduped by IDs.

Local or API
