About mu3u

  • Supports classic .m3u (UTF-8) and HLS-style .m3u8 exports.
  • Understands #EXTM3U header and #EXTINF:duration, title lines.
  • Free-form attribute string per item (e.g., tvg-id, logo, group-title).
  • Drag to reorder (↑/↓), deduplicate by URL, and validate basic fields.
  • All processing happens locally in your browser (no network calls).
