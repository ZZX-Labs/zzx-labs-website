Projects

This is the master index of all projects developed by ZZX-Labs and Archemedean Laboratories—ranging across cybersecurity, AI, Bitcoin infrastructure, creative technologies, metadata systems, cryptographic toolkits, and scientific research.

Each project has its own dedicated page with documentation, source code, architecture diagrams, and usage instructions. These efforts are part of our long-term vision to build sovereign, open, and resilient technology for a post-surveillance world.

CyberChefKit

A locally hosted, GPG-integrated CyberChef fork used in threat analysis, code obfuscation, and encryption workflows.

  • CyberChefKit

    • MetaTagDB

    A distributed metadata hash database for digital forensics, reverse image analysis, and forensic cataloging of leaked files.

  • MetaTagDB

    • Speciedex

    An advanced biological and mythological species taxonomy engine with AI-augmented classification systems for research, art, and genomics.

  • Speciedex

    • SpeciedexCore

    LLM-enhanced inference kernel for species ID, evolutionary mapping, and myth-based creature design integrated with tokenized outputs.

  • SpeciedexCore

    • SpeciedexExplorer

    GUI and visualization layer to navigate the full Speciedex network, filter by trait, era, mythos, genetics, or fictional species groupings.

  • SpeciedexExplorer

    • SpeciedexNet

    Federated peer-to-peer database for decentralized knowledge sharing of flora/fauna/genetic hybrids, using versioned contributions and signed metadata.

  • SpeciedexNet

    • GP (Glyph Protocol)

    A symbolic language and encryption model using glyph-based syntax for obfuscated computation and encrypted digital expression.

  • Glyph Protocol

    • CGAI (Cryptographic Generative AI)

    Secure generative models trained using adversarial data pipelines, including audio, visual, and linguistic creativity under GPG-verified constraints.

  • CGAI

    • GGAI (Generative GAN AI)

    Experimental GAN stacks built for adversarial mimicry, deepfake defense, and cryptographic fingerprinting in media authentication pipelines.

  • GGAI

    • Bit-Tick

    A globally weighted, real-time BTC ticker with microsecond update intervals. Supports price charts, holding estimators, and USD converters.

  • Bit-Tick

    • ISRN (Intelligence Signal Relay Network)

    A covert, zero-trust relay and analytics platform for parsing, tagging, and rebroadcasting network traffic through encrypted tunnels.

  • ISRN

    • ISVN (Intelligence Surveillance Visualization Network)

    Visual surveillance overlay and SIGINT dashboard connected to the ISRN layer. Offers stream replays, anomaly detection, and geoparsed threat feeds.

  • ISVN

    • 4DV

    Temporal-visual modeling platform for four-dimensional simulations, memory mapping, data compression, and creative data sculpture.

  • 4DV

    • Zee-Tu

    A digital persona simulator for constructing sentient-facing AI agents that operate behind sovereign identity stacks (GPG, Bitcoin, etc).

  • Zee-Tu
