SpeciedexExplorer (Web)

SpeciedexExplorer is the web GUI layer for the Speciedex ecosystem. It provides an interactive taxonomy browser and research console that plugs into the Speciedex data model, leveraging SpeciedexCore for inference and SpeciedexNet for federated sync.

Think of it as the human interface to a living knowledge graph of biology and myth—designed for research, conservation, worldbuilding, education, and creative tooling.

Key Capabilities

  • Interactive navigation from kingdom → … → subspecies → cultivars/strains
  • Faceted search by traits, habitat, era, lineage, and mythological context
  • AI-assisted hints and relatedness suggestions via SpeciedexCore
  • Offline-first datasets with signed updates via SpeciedexNet
  • Export packs for research pipelines and creative tooling
  • GPG-signed contribution flow (moderated merges)

How It Integrates

  • Speciedex (Software): canonical schema + ingestion/validation. Explorer reads the normalized graph and renders domain-friendly views.
  • SpeciedexCore (Software): inference kernel for suggestions and trait correlations. Explorer calls Core for related species, likely ancestors, and clustering.
  • SpeciedexNet (Software): federated P2P layer for dataset sync and signed contributions. Explorer can track remotes and pull changes securely.
  • speciedex.org (Web): public portal that mirrors stable releases and provides open read access for non-auth users. Explorer may deep-link to public entries where relevant.
