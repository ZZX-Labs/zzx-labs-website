speciedex.org

speciedex.org is the public-facing portal of the Speciedex project. It provides open access to curated biological and mythological species datasets, taxonomies, and AI-augmented exploration tools.

This web node acts as a living reference library: bridging research, art, genetics, mythology, and conservation datasets under a single sovereign taxonomy.

Key Features

  • Interactive taxonomy browser (kingdom → subspecies → cultivars/strains)
  • AI-augmented classification and trait inference
  • Cross-links to conservation and mythological databases
  • Public knowledge contribution with GPG-signed submissions
  • Export options for research and creative projects
