Designing ZZX-Labs

This page outlines the artistic and engineering process used to craft the visual identity, color palette, grid layout, and frontend/backend architecture of zzx-labs.io. Everything here is intentional: from font kerning to shadow radius.

It also serves as a reference for future brand and UI/UX expansion across our mirror services, onion services, admin panels, and internal tools.

Design Elements & Concepts

Below are the key visual, typographic, and structural components that shape the ZZX-Labs brand identity. Each is interactive, documented, and tied to our style system or `*.json` declarations.

Color Palette

Core brand colors for ZZX-Labs, defined in static/colors.json. All buttons, hover states, and themes derive from this core swatch group.

"primary": "#e6a42b"
"accent": "#c0d674"
"background": "#282828"
"text": "#d1d1d1"

Font Stack

Primary typeface is IBM Plex Mono for code-driven clarity. Titles use the custom Adult Swim font embedded from static/fonts/.

@font-face {
    font-family: 'Adult Swim Font';
    src: url('static/fonts/Adult-Swim-Font.ttf') format('truetype');
}

Navigation Structure

The entire navbar and hamburger menu are handled via responsive CSS flex grid and minimal JavaScript toggling.

<button class="menu-button">☰</button>
<ul class="nav-links">
    <li><a href="/">Home</a></li>
</ul>

Logo System

The ZZX-Labs logo is rendered in two variants: a symbol version (Tibetan Endless Knot) and full logotype. SVG and PNG versions live in static/images/.







Hero Section

Our landing hero uses a semi-transparent black overlay and 8.5rem title font. Fully responsive and animated via scroll events.

.hero {
  background: url('../static/images/hero.jpg') center/cover no-repeat;
  position: relative;
}
.hero-content h1 {
  font-size: 8.5rem;
  color: #c0d674;
}

Animation Classes

Scroll-triggered animations use CSS utility classes. These fire when elements enter the viewport.

.scroll-animation {
    opacity: 0;
    transform: translateY(20px);
    transition: opacity 0.6s ease-out, transform 0.6s ease-out;
}
.scroll-animation.in-view {
    opacity: 1;
    transform: translateY(0);
}

Inspirational Color Grid (Shuffle Algorithm)

Our grid of philosophical influencers uses color classes that are randomly shuffled on load to avoid adjacent duplicates.

// Pseudo-Random Fisher-Yates shuffle
function shuffle(array) {
  for (let i = array.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [array[i], array[j]] = [array[j], array[i]];
  }
}

Icon System

Icons are handpicked SVGs stored in static/icons/. Future support includes custom icon fonts and inline SVG filters.

<img src="../static/icons/lock.svg" alt="Lock Icon" class="icon"/>

Button Classes

All CTA and hero buttons use consistent visual behavior, hover transitions, and responsiveness.

.btn {
  background-color: #e6a42b;
  color: #000;
  padding: 0.8rem 1.5rem;
  border-radius: 4px;
  text-decoration: none;
  transition: background-color 0.3s ease;
}
