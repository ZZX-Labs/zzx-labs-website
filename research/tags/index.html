Research Tags

The Research Tags index provides a topic-based browsing layer for ZZX-Labs papers, reports, analysis, disclosures, datasets, projects, tools, authors, domains, and archive records.

Tags should make research discoverable by subject rather than only by document type. A single tag may connect papers, reports, datasets, tools, project records, disclosures, and analysis items across the site.

Tagging Rules

Recommended Tag Discipline

Use Slugs

Tags should be lowercase, slugified, stable, and predictable, such as bitcoin or machine-learning.

Avoid Duplicates

Do not create redundant tags like ai, artificial-intelligence, and machine-intelligence without purpose.

Use Canonical Names

Keep a canonical tag list so search, filtering, RSS, registry generation, and publication pages remain consistent.

Tag By Meaning

Apply tags based on actual content, not marketing language or speculative association.

Support Cross-Linking

Tags should connect papers, reports, analysis, datasets, projects, disclosures, and tools.

Version Changes

If tags are renamed, preserve aliases or migration notes so older records remain understandable.
