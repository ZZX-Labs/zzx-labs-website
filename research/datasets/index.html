Research Datasets

The ZZX-Labs Research Datasets library indexes datasets used for papers, reports, analysis, disclosures, software experiments, AI/ML work, Bitcoin infrastructure research, OSINT studies, cybersecurity research, technical benchmarking, and reproducible R&D workflows.

Dataset entries should include clear scope, provenance, license, collection method, format, version, hashes, manifests, limitations, and related research records.

Dataset Types

Dataset Collections

Bitcoin Datasets

Node snapshots, mempool records, fee data, mining data, exchange data, Lightning data, and market reference files.

Cybersecurity Datasets

Sanitized logs, indicators, configuration records, defensive telemetry, vulnerability metadata, and security review artifacts.

OSINT Datasets

Public-source records, entity sheets, timeline data, source indexes, citation tables, and structured investigative notes.

AI / ML Datasets

Training, validation, test, labeled, unlabeled, media, text, image, audio, and structured machine-learning datasets.

Benchmark Datasets

Performance measurements, test corpora, comparison records, metrics, hardware results, and software behavior captures.

Archive Datasets

Large collections, historical records, snapshots, mirrored references, research archives, and long-term preservation bundles.

Dataset Status

Publication State

Planned

Dataset concept exists but collection, cleaning, or packaging has not yet started.

Collecting

Raw data is being gathered, measured, scraped lawfully, exported, sampled, or assembled.

Cleaning

Data is being deduplicated, normalized, anonymized, redacted, validated, or converted.

Packaged

Dataset files, manifests, hashes, metadata, README files, and schemas are being prepared.

Published

Dataset has a stable version, download location, manifest, license, and verification records.

Archived

Dataset is preserved for history, superseded, deprecated, retired, or no longer maintained.

Metadata

Recommended Dataset Fields

Identity

Title, slug, subtitle, dataset type, domain, version, authors, contributors, and canonical URL.

Scope

Collection period, geographic scope, included records, excluded records, assumptions, and limitations.

Format

CSV, JSON, JSONL, SQLite, Parquet, TXT, XLSX, images, audio, video, archives, or mixed file trees.

Integrity

SHA256 hashes, manifests, signatures, file counts, size records, schemas, and validation notes.

Licensing

Public, private, commercial, research-only, academic, internal, premium, restricted, or custom terms.

Research Links

Related papers, reports, analysis, disclosures, tools, projects, charts, and downloads.
