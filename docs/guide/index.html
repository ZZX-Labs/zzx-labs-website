User Guide: Navigating ZZX-Labs

This guide outlines how to use the core systems, tools, and features available on the ZZX-Labs website. From API interactions to secure downloads, this page serves as your operational handbook.

What You Can Access

  • Public and private API endpoints with JSON schemas
  • Secure contact forms and messaging systems
  • Custom dashboards, visualizers, and analytic panels
  • Research tools, software, and download interfaces
  • Secure storage, private email, and communications

Tips for First-Time Users

  • Start with the API Index to explore our live endpoints
  • Use /blog/rss to subscribe to updates
  • Visit /downloads for tools and datasets
  • Use the site on a secure browser (Tor, Firefox hardened, Brave)

For Advanced Users

  • Integrate our Bitcoin pricing feeds in your dashboard or CLI
  • Use our GPG tools to interact with ZIRA AI securely
  • Access CLI APIs via /api/cli for terminal-based workflows
  • Host a mirror of this site using Docker & GitHub Pages

All systems are maintained with privacy and resilience as core design goals. For more advanced operations, please consult the System Manual or contact the team.
