Deployment

You can deploy the full ZZX-Labs platform on a secure local server, Raspberry Pi cluster, or cloud instance using provided Docker Compose templates. Full API functionality is accessible via HTTP(S), with real-time logging and visual dashboards.

Manual Format

This manual mirrors UNIX documentation style but is optimized for the web. It is updated live with each versioned release and maintained in Markdown, HTML, and PDF forms.
