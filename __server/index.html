Host Service Reference

Reference service used for the Bitnodes crawler daemon on a Linux host.

[Unit]
Description=ZZX-Labs Bitnodes Crawler Daemon
After=network-online.target redis-server.service
Wants=network-online.target

[Service]
Type=simple
WorkingDirectory=/srv/zzx-labs
ExecStart=/usr/bin/python3 /srv/zzx-labs/tools/bitnodes/bitnodesd.py run
ExecStop=/usr/bin/python3 /srv/zzx-labs/tools/bitnodes/bitnodesd.py stop
Restart=always
RestartSec=10
User=zzx
Group=zzx
Environment=PYTHONUNBUFFERED=1
Environment=REDIS_SOCKET=/tmp/redis.sock
EnvironmentFile=-/etc/zzx-labs/bitnodes.env

[Install]
WantedBy=multi-user.target

Deployment Boundary

Rules for keeping `__server/` useful while preserving operational safety.

Static Public Repo

Only publish frontend-safe docs, templates, examples, and non-secret config references.

No Production Secrets

Keep `.env`, credentials, private keys, SSH material, tokens, and private host configs off-repo.

Mirror-Aware Design

Document local nginx, Raspberry Pi, Dell server, GitHub Pages, and future Flask deployment paths.

API-Ready Later

Use this portal as a bridge toward Flask endpoints, Bitnodes daemon status, and authenticated tools.
