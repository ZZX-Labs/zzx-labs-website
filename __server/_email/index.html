Email Policy

Rules for safe public documentation and future private email service deployment.

Encrypt Sensitive Mail

Use GPG for sensitive inbound and outbound communications wherever practical.

Rotate Keys

Track public key changes, expiration dates, revocations, and signed transition notices.

Separate Public and Private Aliases

Use clean boundaries between public contact, internal ops, client work, alerts, and legal mail.

No Secrets in Git

Never commit bridge credentials, mailbox passwords, tokens, recovery codes, DKIM private keys, or raw mail.
