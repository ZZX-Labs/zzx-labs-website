SFTP Policy

Operational rules for safe future deployment and public repository documentation.

SSH Keys Only

Prefer SSH key authentication with passphrases, rotation, expiration, and revocation records.

Chroot Users

Restrict users to isolated directories and prevent lateral access between clients or partners.

Quarantine Uploads

Stage inbound uploads before publication, scanning, review, archival, or production use.

No Secrets in Git

Never commit usernames, private hostnames, SSH material, access logs, tokens, or client paths.
