Operational Notes

Deployment rules for keeping nginx useful, safe, and mirror-friendly.

Static First

Serve the GitHub Pages-compatible site directly from the repo root whenever possible.

Proxy APIs Only

Keep Flask behind nginx and expose it through `/api/` instead of direct public backend ports.

Cache Public Assets

Cache fonts, CSS, JavaScript, images, and stable JSON while avoiding stale live API outputs.

Deny Secrets

Block dotfiles, `.env`, database files, backups, private logs, and generated secrets.
