Mirror Purpose

ZZX-Labs Bitnodes Mirror

This mirror is an independent frontend mirror and research-oriented reconstruction intended to preserve, study, and extend Bitcoin network visibility using locally generated static JSON snapshots, public frontend pages, and sovereign crawler infrastructure.

Extended Mirror Functionality

Frontend Reconstruction

ZZX-Labs R&D extends the mirror with its own frontend styling, static JSON publication model, local crawler export assumptions, Bitcoin website integration, widget compatibility, and expanded telemetry pages.

Telemetry Expansion

The mirror includes additional views for countries, cities, ASNs, Tor, latency, propagation, agents, ports, versions, service bits, peer health, reachable-state history, and crawler source comparison.
