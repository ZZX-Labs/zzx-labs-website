ZZX-Labs / Bitcoin Network Intelligence

ZZX Bitnodes Mirror

ZZX Bitnodes is a Bitcoin network intelligence platform developed by ZZX-Labs R&D. Inspired by and fully compatible with the original Bitnodes project created by Addy Yeow, the platform provides researchers, node operators, developers, miners, analysts, and educators with detailed visibility into the global Bitcoin peer-to-peer network.

The platform continuously crawls reachable Bitcoin nodes, collects network telemetry, resolves geographic and ASN information, evaluates node health, and presents the resulting datasets through interactive dashboards, maps, rankings, statistics, and APIs. Every Bitnodes page within this section of the site is powered from this central data infrastructure.

Original Bitnodes

A preservation-focused implementation designed to replicate the behavior, structure, datasets, and experience originally provided by Addy Yeow's Bitnodes platform.

This mode maintains compatibility with historic Bitnodes workflows, research projects, scripts, dashboards, and data consumers that expect the traditional Bitnodes format.

Hybrid API

The Hybrid API merges data collected by both the ZZX crawler infrastructure and Original Bitnodes-compatible datasets into a unified interface.

This provides maximum coverage, redundancy, validation, and historical continuity while exposing a single API surface for applications, dashboards, and automated research systems.

Third-Party Compatible Instances

The platform supports external Bitnodes-compatible APIs and independently hosted crawler networks that implement Bitnodes data structures and endpoint conventions.

Organizations, universities, mining groups, exchanges, hosting providers, and researchers can deploy their own infrastructure while maintaining interoperability with the ZZX Bitnodes user interface.

Network Intelligence Capabilities

ZZX Bitnodes collects and analyzes information relating to Bitcoin's publicly reachable peer-to-peer network. Available datasets and visualizations include node counts, countries, cities, autonomous systems, network versions, user agents, latency measurements, propagation observations, uptime estimates, DNS seeder activity, listening ports, ranking systems, peer-health indicators, and historical snapshots.

Additional research efforts are focused on improving network attribution techniques through infrastructure fingerprinting, ASN enrichment, geospatial analysis, VPN and proxy probability scoring, I2P support, Tor visibility improvements, and advanced visualization engines for large-scale Bitcoin network exploration.

Platform Architecture

Data originates from distributed crawler infrastructure operated by ZZX-Labs R&D. Nodes are discovered, validated, enriched, stored, processed, and published through JSON APIs consumed by the Bitnodes frontend. The system is designed to support local mirrors, research deployments, private infrastructure, public web instances, and future federated Bitnodes-compatible networks.

All Bitnodes subpages available within this section—including Maps, Network Map, Nodes, Countries, Cities, ASNs, Tor, Rankings, Peer Health, Uptime, Ports, Agents, Versions, Latency, DNS Seeders, and Propagation—draw their data from the source selected above.
