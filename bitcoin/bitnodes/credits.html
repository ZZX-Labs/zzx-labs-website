Original Bitnodes Project

The original Bitnodes project was created by Addy Yeow. ZZX-Labs R&D gives full credit to Addy Yeow for the original Bitnodes concept, public Bitcoin network crawler, reachable-node snapshot model, API structure, leaderboard concepts, DNS seeder health views, ASN health views, live map, network map, and related Bitcoin node intelligence tooling.

This ZZX-Labs Bitnodes Mirror is an independent frontend mirror and research-oriented reconstruction intended to preserve, study, and extend Bitcoin network visibility using locally generated static JSON snapshots, public frontend pages, and sovereign crawler infrastructure.

This mirror is not intended to erase authorship, obscure origin, or claim invention of the Bitnodes project. It is intended to preserve the functionality, provide proper credit, and adapt the concept for ZZX-Labs’ own Bitcoin network intelligence stack.

ZZX-Labs Additions

ZZX-Labs R&D extends the mirror with its own frontend styling, static JSON publication model, local crawler export assumptions, Bitcoin website integration, widget compatibility, and expanded telemetry pages such as countries, cities, ASNs, Tor, latency, propagation, agents, ports, versions, and peer-health analysis.
