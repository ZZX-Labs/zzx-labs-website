Recommended Usage

Integration Notes

Source Scoped APIs

Prefer api/zzxbitnodes/*.json, api/originalbitnodes/*.json, and api/aggregate/*.json for new integrations.

Static JSON

All API endpoints are static JSON resources suitable for GitHub Pages deployment and offline mirroring.

Maps

Maps and live maps should consume generated map datasets instead of parsing complete node registries.

Registry Backups

Large chunked registry backups remain separate from the public API endpoints and are maintained by dedicated workflows.
