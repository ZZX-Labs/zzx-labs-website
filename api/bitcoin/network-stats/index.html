Bitcoin Network Stats API

This API provides direct access to real-time Bitcoin blockchain statistics sourced from ZZX-Labs’ fully self-hosted Bitcoin Core full nodes. It offers live data on the mempool, block propagation, fee rates, and unconfirmed transaction queues.

The data updates every 250ms and is ideal for mempool monitoring, dynamic fee estimation, and network health visualizations. No third-party services are used—fully sovereign, fully private.

Public Endpoint

GET /api/bitcoin/network-stats

Returns real-time mempool size, recommended fees, unconfirmed transactions, and current block stats.
