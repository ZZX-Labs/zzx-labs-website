Bitcoin Price API

This API provides the globally weighted Bitcoin spot price (BTC/USD), updated every 250ms based on a volume-weighted average from 100+ live exchanges.

Data is calculated locally using ZZX-Labs' own fetcher and is not reliant on any third-party aggregators. Response format is application/json.

Endpoint

GET /api/bitcoin/price

Returns the current BTC/USD weighted price.

Example Output

{
  "price": 69234.85,
  "currency": "USD",
  "timestamp": "2025-05-01T18:27:10.250Z",
  "source_count": 102,
  "volume_weighted": true
}

Usage

You can fetch data using curl or from a Python/JavaScript client.

curl https://zzx-labs.io/api/bitcoin/price
