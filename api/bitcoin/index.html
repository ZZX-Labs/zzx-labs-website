Bitcoin API Suite

The Bitcoin API Suite provides direct access to real-time pricing, on-chain statistics, Lightning Network data, wallet interactions, and node-level analytics. These endpoints are foundational to all Bitcoin-based operations across ZZX-Labs platforms.

APIs are updated at sub-second intervals, sourced from over 100 global exchanges, and calculated using volume-weighted ratios. Data integrity is ensured via self-hosted Bitcoin Core nodes and GPG-signed responses.

Live Price API

Returns the globally averaged BTC/USD spot price calculated every ¼ second from the top exchange feeds by volume.

  • /api/bitcoin/price

    • Bitcoin Calculator API

    Convert any amount of Bitcoin between units (BTC, mBTC, μBTC, sat, msat, μsat) or to USD. Supports JSON and CLI formatting.

  • /api/bitcoin/calculator

    • Tracker API

    Track Bitcoin portfolio values in real-time. Combine holdings with live pricing to return total fiat or BTC-equivalent value.

  • /api/bitcoin/tracker

    • Converter API

    Flexible conversion between fiat and Bitcoin units, or across BTC sub-denominations. Supports country-specific currency codes (ISO-4217).

  • /api/bitcoin/converter

    • Bitcoin Charts API

    Returns JSON chart data for time-series Bitcoin metrics: price, volume, hash rate, mempool size, block times, and more.

  • /api/bitcoin/chart

    • Market Stats API

    Real-time market data across all major BTC exchanges. Includes trading volume, spread, depth, and arbitrage alerts.

  • /api/bitcoin/market-stats

    • Network Stats API

    Live blockchain data from our full nodes. Returns mempool activity, difficulty, fees, transaction sizes, and confirmations.

  • /api/bitcoin/network-stats

    • Mining Stats API

    Track hashrate, mining pool distribution, orphan rate, reward stats, and fee-to-subsidy ratio across the network.

  • /api/bitcoin/mining-stats

    • Lightning Network Stats API

    Get capacity, channel count, node count, and route data for LN. Useful for routing, liquidity tracking, and LN research.

  • /api/bitcoin/ln-stats

    • Bitcoin Explorer API

    Explore blocks, transactions, and addresses with detailed metadata returned in JSON. Self-hosted, privacy-respecting backend.
