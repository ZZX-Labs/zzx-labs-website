Bitcoin Mining Stats API

Monitor the heartbeat of Bitcoin’s mining infrastructure with live data pulled from ZZX-Labs' synchronized node indexers and mining pool scanners. This API delivers up-to-date hashrate estimates, orphaned block tracking, reward details, and fee-to-subsidy ratios.

All data is privacy-respecting and generated in-house without reliance on third-party APIs. Use this for mining dashboards, pool telemetry, or economic modeling of network security.

Public Endpoint

GET /api/bitcoin/mining-stats

Returns recent hashrate, block rewards, orphan rate, and fee-based mining stats.

  "timestamp": "2025-05-01T12:00:00Z",
  "estimated_hashrate_THs": 495200,

  "fee_to_subsidy_ratio": 6.94,

}

Key Metrics

  • Real-time network hashrate estimate
  • Current subsidy and average fees per block
  • Pool dominance estimation (via block template tracking)
  • Orphan rate analytics per rolling 24h
