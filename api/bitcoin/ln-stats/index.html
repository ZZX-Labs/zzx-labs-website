Lightning Network Stats API

Access real-time data from the Bitcoin Lightning Network including total capacity, node count, active channel count, and route propagation efficiency. This API supports routing intelligence, capacity analytics, and uptime evaluation.

ZZX-Labs operates non-custodial LN nodes to feed this telemetry. Outputs are useful for protocol research, liquidity optimization, and public LN monitoring dashboards.

Public Endpoint

GET /api/bitcoin/ln-stats

Returns current network stats for Lightning including total BTC locked, node & channel counts, and connectivity.

Sample Response

{
  "timestamp": "2025-05-01T12:00:00Z",
  "total_capacity_btc": 5763.84,
  "total_channels": 83541,
  "active_nodes": 19145,
  "avg_channel_capacity_btc": 0.069,
  "avg_channel_age_days": 198
}
