Bitcoin Price Charts

This module renders dynamic, real-time Bitcoin price charts powered by ZZX-Labs’ internal API and stored historical price data from major global exchanges. The system supports multiple timeframes (1m, 5m, 1h, 1d, 1w, 1m), zoom, pan, and fiat overlays.

Charts can be embedded in dashboards or served as SVG/PNG images for static output. The charting engine is private, lightweight, and zero-dependency—ideal for air-gapped systems or hardened admin dashboards.

Charting API

GET /api/bitcoin/charts?interval=1m

Returns timestamped price points for Bitcoin. Supports ?interval=1m/5m/1h/1d.

Visual Output Modes

  • SVG / PNG static exports
  • Canvas / Chart.js live streaming
  • Terminal ASCII chart rendering (CLI)
