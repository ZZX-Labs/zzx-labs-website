Market Stats API

The Market Stats API aggregates real-time order book data, trade volume, spreads, slippage metrics, and arbitrage windows across all tracked exchanges. Powered by ZZX-Labs’ full aggregation node, this API offers high-resolution statistics updated every 250ms.

Use it to build dashboards, trading algorithms, arbitrage scanners, or exchange health monitors. All data is returned in clean JSON format with optional filters per exchange or fiat pair.

Public Endpoint

GET /api/bitcoin/market-stats

Returns exchange volumes, spread, price depth, slippage bands, and price leaderboards.
