Bitcoin Explorer API

Query individual Bitcoin blocks, transactions, or addresses via our fast, self-hosted explorer backend. This zero-trust REST API enables metadata-rich blockchain exploration without third-party analytics or tracking.

All queries are sandboxed, privacy-respecting, and powered by ZZX-Labs' full archival nodes and indexed databases. Built for developers, researchers, and forensic analysts.

Block Lookup

GET /api/bitcoin/explorer/block/:block_hash

Returns full block metadata including height, time, size, tx count, Merkle root, and raw transaction hashes.

Transaction Lookup

GET /api/bitcoin/explorer/tx/:txid

Returns transaction details such as input/output structure, size, fee, witness data, and confirmations.

Address Lookup

GET /api/bitcoin/explorer/address/:btc_address

Returns UTXO set, balance, transaction history, and last activity for the specified address.
