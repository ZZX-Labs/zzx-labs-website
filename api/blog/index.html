Blog API

The Blog API suite powers the publishing, syndication, and secure administration of all ZZX-Labs blog content. It supports public access via RSS, developer access via JSON feeds, and encrypted backend post submission for approved authors.

Blog posts are written in plaintext `.txt` format, parsed and converted to semantic HTML on-the-fly. Metadata including tags, timestamps, and categories are embedded and published with each entry.

RSS Feed Endpoint

Publicly accessible RSS/Atom XML feed. Used to syndicate ZZX-Labs blog posts to readers, bots, and aggregators.

  • /api/blog/rss

    • Public Blog Feed API

    Returns a JSON feed of published blog content for integration with dashboards, custom frontends, or CLI-based readers.

  • /api/blog/feed

    • Blog Post Submission API

    Private admin-only API allowing authorized team members to upload blog posts as `.txt` files. Automatically timestamped, parsed, converted to HTML, and indexed.

  • /api/blog/submit

    • Tagging & Metadata API

    Retrieve metadata, tags, category lists, author details, and post frequency stats. Useful for analytics dashboards and topic-based sorting.

  • /api/blog/tags

    • Post Search API

    Search all blog posts by title, tag, content, or metadata field. Returns matched post excerpts and full content on demand.

  • /api/blog/search

    • Want to Publish on Our Blog?

    Contributors, guest writers, and community members are invited to submit blog content through our admin-controlled API. GPG-signed identity and approval required.
