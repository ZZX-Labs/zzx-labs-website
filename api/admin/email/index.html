Secure Email Server API

This API interfaces with ZZX-Labs' internal ProtonMail-compatible GPG-based email server. It offers administrative functionality for managing encrypted mailboxes, access control, user provisioning, and email routing within a zero-trust encrypted infrastructure.

Core Functionalities

  • Account Management: Create, revoke, or rotate GPG-protected accounts.
  • Inbox Access: Admin-level access to encrypted inbox metadata (headers only, content stays encrypted).
  • IMAP Control: Enable/disable IMAP access for specific users or teams.
  • GPG Key Management: Upload and verify user public keys for email exchange.
  • Audit & Delivery Logs: Query logs of deliveries, failures, and authentication events.

Create Email Account

Provision a new internal account for encrypted email communication.

{
  "cmd": "create_account",
  "user": "alice@zzx-labs.io",
  "pubkey": "-----BEGIN PGP PUBLIC KEY BLOCK----- ...",
  "auth_token": "admin-tier-token"
}

Toggle IMAP Access

Enable or disable IMAP/SMTP relay capabilities for a specific internal account.

{
  "cmd": "set_imap",
  "user": "alice@zzx-labs.io",
  "status": "enabled",
  "auth_token": "admin-gpg-token"
}
