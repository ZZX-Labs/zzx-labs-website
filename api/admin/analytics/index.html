Analytics Engine API

The Analytics API provides a secure endpoint for querying, monitoring, and exporting real-time data about ZZX-Labs platform usage. It supports both passive telemetry (e.g., time-on-page, referral paths) and active event logging (e.g., API calls, clicks, scroll behavior).

Designed for privacy-respecting internal analytics, all data is stored locally, not sent to third parties. ZIRA may also integrate directly to summarize trends and surface anomalies.

Core Capabilities

  • Live Stats: Track real-time metrics like active sessions, CPU usage, or total API requests.
  • Session Histories: Retrieve detailed logs by IP, session token, or user class.
  • Custom Dashboards: Build personalized analytics panels via API queries + ZIRA widgets.
  • Privacy-Respecting: No cookies or third-party beaconsâ€”analytics stay within your node.
  • Export JSON Reports: Download structured data reports for external audits or visualization.

Query Active Sessions

Return the count and metadata of currently active user or system sessions.

{
  "action": "get_sessions",
  "auth_token": "gpg-auth-signed"
}

Analytics Snapshot

Fetch a quick overview of live metrics for admin dashboards and ZIRA feeds.

{
  "metrics": [
    "active_users", 
    "avg_time_on_page", 
    "api_requests_24h"
  ]
}

Download Audit Report

Generate and download a timestamped analytics report in machine-readable JSON.

{
  "report": "site_analytics_q2.json",
  "include": ["sessions", "API calls", "referrals"]
}
