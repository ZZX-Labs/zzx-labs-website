Security Dashboard API

The Security API provides real-time access to behavioral threat data, tiered actor classifications, incident logs, and system alert telemetry. This endpoint is designed for defensive monitoring and proactive analysis of suspicious or adversarial activity across the ZZX-Labs ecosystem.

All security telemetry is streamed securely, logged with cryptographic integrity, and gated by role-based access control with GPG identity verification.

Capabilities

  • Tier Classification: Track IPs, accounts, devices, and bots across threat levels (Tier 0–5).
  • Live Alert Triggers: Webhooks or streaming feeds for specific behavioral or risk-based alerts.
  • Threat Attribution: Profile adversarial entities by request pattern, probe signature, or behavioral fingerprint.
  • Secure Incident Log Access: All security logs are cryptographically hashed and timestamped per incident.
  • ZIRA Alert Relay: Stream suspicious activity logs directly to ZIRA for NLP classification or dashboard output.

Check Threat Tier

Query an IP, session, or device fingerprint and retrieve associated threat tier and recent actions.

{
  "action": "check_tier",
  "target": "192.168.0.11",
  "auth": "gpg-signed-token"
}

Fetch Alerts

Retrieve triggered security events over the last hour or specified interval. Stream-ready or batched.

{
  "action": "fetch_alerts",
  "interval": "1h",
  "priority": "high",
  "verify": true
}
