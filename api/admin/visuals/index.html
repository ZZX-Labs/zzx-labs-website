Data Visualization Dashboard API

This endpoint powers all graphical and data-driven visual interfaces across ZZX-Labs dashboards and admin terminals. Whether you're plotting site analytics, transaction flows, or behavioral metrics, the Visuals API delivers live, customizable, and streamable datasets.

Output can be returned as JSON for frontend rendering, or passed through ZIRA for NLP summarization and visual inference.

Capabilities

  • Custom Graphs: Submit JSON data and config for rendering charts, heatmaps, or time-series graphs.
  • Real-Time Streams: Visualize API usage, system health, or Bitcoin market data in real time.
  • Dashboard Widgets: Dynamically insert or control HTML/JS visual widgets via secure API calls.
  • Snapshot Export: Export rendered graphs as SVG, PNG, or JSON configs.
  • Secure Access: GPG authentication required for modifying live dashboards or injecting visual content.

Render Line Chart

Submit a dataset and config to render a chart in the admin dashboard visual area.

{
  "type": "line",
  "dataset": "btc_price_stream",
  "time_range": "24h",
  "auth": "gpg-signed-token"
}

Dashboard Inject Widget

Inject a visual widget into a live dashboard. All widget JS is sandboxed and verified by hash.

{
  "cmd": "inject_widget",
  "widget_id": "btc_live_ticker",
  "dashboard": "main",
  "signature": "verified-user-gpg"
}
