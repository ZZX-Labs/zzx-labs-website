CyberChef API

The CyberChef API at ZZX-Labs provides local, self-hosted access to a secure version of GCHQ's CyberChef tool—offering encoding, decoding, hashing, encryption, and data transformation functions via API.

Ideal for forensic workflows, cryptography pipelines, binary transformations, steganographic operations, or embedded analysis inside broader administrative flows. All operations are executable via CLI, JSON RPC, or web dashboard.

Available Operations

  • Encoding/Decoding: Base32, Base58, Hex, URL, Morse, ASCII/UTF8, Unicode, ROT, etc.
  • Cryptographic Hashing: SHA1, SHA2, SHA3, MD5, RIPEMD160, Whirlpool, BLAKE2, etc.
  • Encryption/Decryption: AES, XOR, GPG-compatible flows, XOR key rotation.
  • Binary Tools: XOR Brute Force, Byte Rotation, Bit Shifting, Visual XOR Diffing.
  • Visual/Hex: Image decoding, hex viewers, character maps, dump renderers.

Base32 Encode/Decode

CyberChef supports a wide range of base encodings. Use via:

{
  "operation": "Base32 Encode",
  "input": "bitcoin forever",
  "output_format": "text"
}

GPG-compatible Encryption

Encrypt payloads for secure transmission and command relay:

{
  "operation": "AES Encrypt",
  "input": "classified data",
  "key": "zzxpassphrase",
  "mode": "AES-GCM"
}

Hash Verification

Generate and compare cryptographic checksums and identity hashes:

{
  "operation": "SHA3-256",
  "input": "zzx-labs"
}
