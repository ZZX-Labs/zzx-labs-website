Video Stream API

The Video Stream API provides secure access to live, looped, or archived video feeds from ZZX-Labs infrastructure. Feeds are used in surveillance terminals, multimedia dashboards, livestream applications, and data visualization systems.

All streams are authenticated by access tier. Live feeds require stream keys, while archived footage is accessed via hashed tokens and time-based URLs. Public demo streams are available via the endpoints below.

Capabilities

  • Live Streaming: Secure live video feeds broadcast from IP cams or virtual media nodes.
  • Archived Playback: MP4/OGG content from static/video/videos/ made available for download or streaming.
  • Overlay Support: Stream layers can support captions, frame overlays, or visual indicators.
  • Authenticated Access: Access granted via session keys, tokens, or badge profile tiers.
  • Real-time Dashboard Feed: Integrated into admin terminals or visual data panels.

Live Feed Endpoint

Access an MJPEG, WebM, or HLS stream of a live feed via the secure API. Replace the key with an authorized stream token.

curl https://zzx-labs.io/api/media/video/stream?key=your-stream-key

Archived Video Access

Stream or download pre-recorded MP4s stored in /static/video/videos/. Encrypted filenames may require decoding or session token unlock.

    Embed content in dashboards or visual panels. Supports autoplay, loop, mute, and frame overlays for training mode or analysis mode.

</video>
