API Suite

Welcome to the API Gateway of ZZX-Labs. This portal offers access to our full catalog of public, private, and internal-use APIs powering the ZZX-Labs ecosystem—from Bitcoin price feeds and payment endpoints to blog publishing, research collaboration, and administrative control.

All APIs follow zero-trust design principles, with fine-grained access control, GPG-authenticated endpoints, and rate-limiting to ensure stability and privacy. Public endpoints are freely accessible, while private and administrative interfaces require elevated credentials.

All APIs: Public & Private

The following API endpoints are maintained and updated by ZZX-Labs. Each API supports machine-readable output, token-based authorization (or GPG handshake), and integration with CLI tools or Python clients. Documentation and JSON schema are available per endpoint.

Bitcoin Price API

Provides a globally weighted Bitcoin spot price (BTC/USD) calculated from volume-weighted price quotes across 100+ real-time exchange feeds. Updates every ¼ second.

  • Bitcoin Price

    • Payments API

    Handles all service and shop payments via Bitcoin only. A new on-chain or Lightning address is generated per transaction using ZZX-Labs’ own full node. Supports invoices, receipts, and transaction tracking.

  • Payments API

    • Blog API

    Public RSS feed for blog content consumption, and a private endpoint for posting `.txt` blog entries that auto-render to HTML. Includes timestamping, metadata, and RSS auto-sync.

  • Blog API

    • Downloads API

    Public access to software, papers, tools, charts, datasets, and diagrams. Private endpoint allows secure uploads with metadata, tags, and embedded thumbnails.

  • Downloads API

    • Research API

    Restricted-access API for peer researchers with approved credentials. Allows secure uploads and downloads of datasets, whitepapers, models, and experimental tools across shared projects.

  • Research API

    • Secure Communications API

    Send and receive GPG-encrypted instant messages, emails, SMS texts, and voice calls (via text-to-speech synthesis). Designed as a sovereign, self-hosted messaging stack for ZZX-Labs and its partners.

  • Secure Comms API

    • Admin API Suite

    Private admin-layer API granting full control over system dashboards, user management, analytics, blog moderation, secure file servers, and command line integrations. Access controlled via cryptographic tokens and internal role hierarchy.

  • Admin API

    • ZIRA Assistant API

    Backend AI assistant interface for ZIRA, the Zero-Trust Interface ReconAdmin. Query internal datasets, automate dashboards, interact with logs, and issue secure commands through ZIRA’s encrypted model interface.

  • ZIRA API

    • CyberChef API

    Interface for encoding, decoding, hashing, and data transformation using a secure CyberChef instance integrated into the ZZX-Labs admin toolchain.

  • CyberChef API

    • RSS Feed API

    Serves RSS feed endpoints for blog posts, whitepapers, public releases, and changelogs. Automatically updated with each push to the live documentation or blog system.

  • RSS Feed API

    • CLI API (cmder integration)

    Enables remote CLI execution and monitoring using ZZX-Labs’ cmder shell environment. Integrated with audit logging and role-based command constraints for high-security use.

  • CLI API

    • Inspirational Quotes API

    Pulls curated quotes from figures featured in our Inspiration page (e.g., Schrodinger, Diffie, Hellman, Curtis). Used to inject signal into dashboards, ZIRA responses, CLI terminals, and analytic panels.

  • Quotes API

    • Easter Egg API

    Delivers encrypted responses to special triggers, hidden commands, or specific visitor behaviors. Often used in cyber training scenarios, badge-based access games, or as part of hidden system flows.

  • Easter Eggs API

    • Video Stream API

    Provides access to local or live-streamed camera/video feeds from ZZX-Labs infrastructure. Also used to deliver multimedia overlays and real-time footage to admin dashboards or visualizer terminals.

  • Video Stream API

    • Audio Stream Visualizer API

    Delivers real-time audio data visualizations from mic or line-in sources. Integrated with VLC Box, visualizers (e.g. projectM), and playback systems for monitoring, alerts, or presentation environments.

  • Audio Stream API

    • SFTP Server Control API

    Admin-only API to manage, mount, and interface with ZZX-Labs' local SFTP server. Provides file browsing, permission control, user credential management, and remote archive deployment.

  • SFTP Control

    • Secure Email Server API

    Controls internal ProtonMail-compatible GPG email stack. Allows account creation, mailbox config, IMAP access control, and integration with ZIRA and secure messaging functions.

  • Email Server API

    • Secure Storage Server API

    Admin endpoint for configuring and encrypting local file storage with LUKS/GPG/AES. Used for mirroring downloads, staging research files, and segmenting critical assets across disk volumes.

  • Storage Server API

    • Data Visualization Dashboard API

    Endpoint for generating and managing all visual analytics interfaces. Feeds data into web dashboards, the Bitcoin analytics panel, and ZIRA’s visualization layer.

  • Visualization API

    • Security Dashboard API

    Provides endpoint access to user behavior logs, Tier actor profiles, alert triggers, threat ratings, and counterintelligence tools. Supports IP/machine/device profiling and session tracing.

  • Security API

    • Recon Dashboard API

    Tracks suspicious users, bots, crawlers, probes, and escalated actors (Tier 1–5). Triggers honeypots, behavior sandboxes, and ZIRA-based analysis on flagged entities.

  • Recon API

    • Analytics Engine API

    Streams site usage statistics, performance metrics, referral paths, time-on-page data, and behavioral analytics to the Admin dashboard or terminal shell. All analytics are privacy-respecting and internally stored.

  • Analytics API

    If you’re a research partner, integrator, developer, or operator working with ZZX-Labs, you may be eligible for secure token access to private and administrative API tiers. Each endpoint is documented and versioned.
