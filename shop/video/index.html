Metadata

Video Metadata and Documentation

Video packages should include metadata, license terms, hashes, manifests, preview files, source notes, and usage restrictions. Research footage should include collection context, location scope, date, and subject notes where appropriate.

Clip Metadata

Title, category, duration, resolution, FPS, codec, format, color profile, date, and tags.

Source Notes

Camera, lens, location scope, environment, subject notes, release notes, and recording context.

License Notes

Personal, commercial, editorial, publication, app/game, enterprise, exclusive, or custom usage rights.

Release Notes

Version, changes, corrections, replacement clips, retired footage, and updated license terms.

Manifests

JSON manifests listing files, previews, formats, sizes, hashes, durations, and license records.

Verification

SHA256 hashes, signatures, official URLs, manifest files, and release records.
