Quality

Icon Package Standards

Consistent Grid

Icon sets should use consistent dimensions, alignment, stroke weight, corner radius, and spacing.

Clear Naming

Icons should use predictable filenames, slugs, tags, categories, and metadata records.

Multiple Sizes

PNG exports should include standard sizes for web, app, favicon, documentation, and preview workflows.

Source Files

Commercial packs may include SVG, AI, PSD, EPS, or other editable source files where licensed.

License File

Every package should include clear license, attribution, redistribution, and commercial-use terms.

Manifest

Icon packs should include a JSON manifest listing files, categories, tags, formats, sizes, and hashes.
