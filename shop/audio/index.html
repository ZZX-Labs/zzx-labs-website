File Formats

Audio File Formats

WAV

Uncompressed audio for production, editing, sound design, testing, datasets, and archival delivery.

FLAC

Lossless compressed audio for archival releases, field recordings, datasets, and high-quality libraries.

MP3

Compressed audio for previews, casual playback, lightweight delivery, and broad device compatibility.

OGG

Open compressed audio for apps, games, web delivery, loops, and open-source workflows.

AIFF

Professional uncompressed audio format for production, editing, and compatibility with audio tools.

OPUS

Efficient compressed audio for speech, streaming, apps, and low-bandwidth delivery.

MIDI

Note data, test sequences, generative music sketches, device triggers, and lightweight music control files.

Archives

ZIP, TAR, TGZ, 7Z, segmented sample packs, metadata bundles, and full audio-library packages.

Metadata

Audio Metadata and Documentation

Professional audio packages should include metadata, documentation, license files, hashes, and clear naming. Research audio should also include collection notes, recording conditions, labels, and dataset structure.

File Metadata

Title, category, duration, sample rate, bit depth, channels, format, tags, and recording date.

Recording Notes

Location, microphone, recorder, environment, weather, room tone, signal chain, and source context.

Dataset Labels

Class labels, tags, taxonomy, transcription notes, segmentation, and annotation records.

Package Manifest

JSON manifest listing files, hashes, durations, categories, formats, licenses, and preview paths.

Licensing Notes

Personal, commercial, publication, app/game, dataset, enterprise, exclusive, or custom usage terms.

Verification

SHA256 hashes, signatures, release notes, version history, and official download references.
