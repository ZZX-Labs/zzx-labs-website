Photography

The ZZX-Labs Photography catalog contains ZZX-Labs-owned stock imagery, field photography, technical photography, research imagery, infrastructure photography, nature photography, urban photography, documentary images, workstation images, product images, and still-image assets for lawful use in media, websites, publications, presentations, apps, reports, books, and design work.

This section is for photography owned, produced, captured, edited, or explicitly licensed for distribution by ZZX-Labs R&D. Third-party images should not be sold here unless ZZX-Labs has clear rights to distribute, sublicense, or package them.

Metadata

Photography Metadata and Documentation

Photography packages should include metadata, license terms, hashes, manifests, preview files, source notes, and usage restrictions. Research photography should include collection context, location scope, date, and subject notes where appropriate.

Image Metadata

Title, category, resolution, format, color profile, camera notes, date, tags, and collection name.

Source Notes

Camera, lens, location scope, environment, subject notes, release notes, and capture context.

License Notes

Personal, commercial, editorial, publication, app/web, enterprise, exclusive, or custom usage rights.

Release Notes

Version, replacements, corrections, retired files, updated licensing, and collection changes.

Manifests

JSON manifests listing files, previews, formats, sizes, hashes, dimensions, and license records.

Verification

SHA256 hashes, signatures, official URLs, manifest files, and release records.
